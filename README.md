# Umap-Utilites
Just a bunch of QoL stuff I made working with Minshu [UmapImporter 2](https://github.com/MinshuG/BlenderUmap2)

This was made for Blender 4.1! I dunno how it will act for older versions of Blender.



# Installation
1. Head to [Releases](https://github.com/jaxbline/Umap-Utilites/releases)
2. Download the latest
3. Open Blender
4. Goto preferences
5. Add-ons
6. Install
7. Install Plugin
8. Profit
